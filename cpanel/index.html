Simple API surface

Use your API key from any app, bot, website, or automation workflow. Base URL: https://api.sms.luffyxd.store/smsapi/v1

Send an SMS
POST /api/sms/send Authorization: Bearer YOUR_API_KEY Content-Type: application/json { "phone_number": "+9876543210", "message": "Hello from XD SMS Gateway", "sim_slot": 1 } // Response { "request_id": "uuid-here", "status": "pending", "message": "SMS queued" }
Bulk send
POST /api/sms/send-bulk Authorization: Bearer YOUR_API_KEY { "messages": [ { "phone_number": "+111", "message": "Hi!" }, { "phone_number": "+222", "message": "Hey!" } ] } // Response { "results": [ { "request_id": "uuid-1", "status": "pending" }, { "request_id": "uuid-2", "status": "pending" } ] }
Android polling (gateway app)
// Device polls for queued messages GET /api/get X-Api-Key: YOUR_API_KEY // Response — message ready to send { "success": true, "id": "queue-item-id", "request_id": "uuid", "phone_number": "+9876543210", "message": "Your OTP is 123456", "sim_slot": 1 } // After sending, report delivery POST /api/done { "id": "queue-item-id", "status": "sent" }
